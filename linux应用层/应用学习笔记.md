# 1、系统头文件   
## 1、`#include <sys/types.h>`   ->定义类型   
其中包括的常见类型有：  
```c
    pid_t    // 进程 ID
    uid_t    // 用户 ID
    gid_t    // 组 ID
    size_t   // 内存/长度
    off_t    // 文件偏移量
    ssize_t  // 有符号 size
```

## 2、`#include <sys/stat.h>`  ->  文件属性
常见内容：   
```c
    struct stat    // 文件信息结构体
``` 
常见函数：
```c
    stat() 
    fstat()
    lstat()
    mkdir()
    chmod()
    umask()
```
常见宏:
```c
    S_IFREG   // 普通文件
    S_IFDIR   // 目录
    S_IRUSR   // 用户读权限
    S_IWUSR   // 用户写权限
```

## 3、`#include <fcntl.h>` -> 打开/控制文件  
常见函数：
```C
    open()
    fcntl()
```
常见宏：
```c
    O_RDONLY
    O_WRONLY
    O_RDWR
    O_CREAT
    O_TRUNC
    O_APPEND
```

## 4、`#include <unistd.h>` -> 读写/进程/系统调用 POSIX操作系统API(重要)   
常见函数:
```c
    read()
    write()
    close()
    lseek()

    fork()
    exec()
    getpid()

    sleep()
    usleep()
```
